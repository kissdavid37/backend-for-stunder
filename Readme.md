## Endpoints

| URL          | HTTP method | Auth | JSON Response  |
| ------------ | ----------- | ---- | -------------- |
| /users/login | POST        |      | user's token   |
| /users       | GET         | Y    | all users      |
| /questions   | GET         |      | all products   |
| /users       | POST        | Y    | new user       |
| /users       | PATCH       | Y    | edit profile   |
| /users       | DELETE      | Y    | Delete profile |
